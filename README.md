# Simple HTTP Server

## Requirements

- GCC compilator
- GNU Make

## Run

1. Build project

```shell
make
```

2. Run

```shell
make run
```

OR

```shell
./build/main
```

3. Rebuild and run

```shell
make all run
```

## Clean

In case if you want to clean your project from generated object files and executables run:

```shell
make clean
```
